Day 3 - CSS Basics

Box Model, Selectors, Colors, Fonts

1. Box Model

Box model has 4 parts: Content, Padding, Border, Margin

This is a box. It has padding, border and margin applied.

padding = space inside the box

border = line around the box

margin = space outside the box

2. Selectors

Element selector targets the HTML tag directly like h1, p, body

Class selector starts with a dot like .section .box-demo

ID selector starts with # like #header #footer

3. Colors

There are 4 ways to write colors in CSS:



Red = color name (red)

Blue = hex code (#3498db)

Green = rgb(39, 174, 96)

Purple = rgba - transparent color

4. Fonts

Big text - font-size: 26px

Small text - font-size: 13px

Bold text - font-weight: bold

Italic text - font-style: italic

Underline text

uppercase text
